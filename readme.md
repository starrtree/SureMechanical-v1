# SURE Group Website

A modern static website for SURE Group, the parent company of SURE Mechanical, Cincinnati Air Conditioning Company, and Thermolinear.

## Commands

```bash
npm run build
```

The build script generates the static routes in `dist/` from reusable data arrays and templates in `src/` and `scripts/`.

## Routes

- `/`
- `/about/`
- `/companies/`
- `/services/`
- `/industries/`
- `/projects/`
- `/careers/`
- `/contact/`
